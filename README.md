# Calidad - Entorno Integración Continua

Proyecto mínimo que sirve un `Hola Mundo` en HTML y tiene una prueba unitaria.

Instalación y uso:

1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar tests:

```bash
npm test
```

3. Ejecutar localmente:

```bash
npm start
# abrir http://localhost:3000
```

4. Despliegue en Vercel:

- Conéctalo a tu repositorio en https://vercel.com y Vercel detectará `vercel.json` y desplegará la función.
- O usa la CLI:

```bash
npm i -g vercel
vercel
```
